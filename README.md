# nuxt-typescript-sample

Nuxtにtypescriptを導入したサンプルです。
手順、説明等は順次更新していきます。(現時点ではまだ未作成)

## Build Setup

``` bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn run dev

# build for production and launch server
$ yarn run build
$ yarn start

# generate static project
$ yarn run generate
```
